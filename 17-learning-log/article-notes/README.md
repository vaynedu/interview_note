# 文章精读笔记

> 读完一篇好文章 / 博客 / 论文之后，按"考题 + 四段式答案"形式消化沉淀。
>
> **目的**：被动阅读 → 主动检验 → 形成肌肉记忆。
>
> **节奏**：每读一篇有价值的文章产出一份。

---

## 文件命名

```
YYYY-MM-DD-<主题缩写>.md
```

例：
- `2026-05-10-xiaomi-distributed-transaction.md`
- `2026-06-15-martin-kleppmann-redlock.md`

## 维护原则

- ❌ 不直接抄文章原文（要有自己的提炼）
- ❌ 不只列题不答（要写完整答案）
- ✓ 每题必带"边界 / 代价"（资深视角）
- ✓ 每题必关联仓库（让笔记成为索引）
- ✓ 一题一答的形式，方便随机抽查复习
