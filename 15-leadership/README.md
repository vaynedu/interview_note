## 核心认知

- **TL 不是技术最强的人**，是让团队最强的人
- **业务理解决定职业上限**，不是技术深度
- **战略 = 决定不做什么**（做减法比做加法难）
- **影响力 = 晋升 P7/P8 硬通货**（不是只看你做了什么）
- **可逆决策快做、不可逆决策慢做**（Bezos 原则）
- **向上管理不是拍马屁**，是双赢
