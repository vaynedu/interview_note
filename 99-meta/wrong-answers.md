## 使用说明

每条按以下模板：

```markdown
## YYYY-MM-DD - <主题分类> - <一句话标题>

**Q**: 完整问题（面试官原话或自己模拟的）

**当时答的**: 我当时怎么答的（哪怕半句话）

**标准答案**: 应该怎么答（简洁 + 可复用）

**为什么错**: 一句话点出根因（知识盲区 / 表达问题 / 取舍没说清）

**关联知识点**: [文件名](../path/file.md)

**类型标签**: #原理 #场景 #取舍 #量化 #实战
```

## 2026-05-10 - Go 并发 - sync.Pool 内部清理时机

**Q**: sync.Pool 里的对象什么时候会被清理？

**当时答的**: GC 的时候清掉一部分

**标准答案**: 每次 GC 开始前，把 victim cache 清空，把 local cache 移到 victim。所以一次 GC 周期内最多保留两轮的对象。

**为什么错**: 知道结论但不知道 victim 机制，被追问"为什么不一次清完"答不上。
