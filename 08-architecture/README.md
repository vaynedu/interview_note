## 设计原则

- **图文并茂**，关键概念用 Mermaid
- **每篇独立可读**，高内聚优先于去重
- **结合真实项目** `ddd_order_example` 举例
- **大厂案例**：阿里 / 美团 / 字节 / 腾讯 / Netflix / Shopify
- **面试题驱动**，每章 10 道高频题 + 加分点
- **架构决策模板**：ADR + 选型框架 + 反模式清单
